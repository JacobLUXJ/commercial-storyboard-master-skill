# 影视分镜 Skill

帮助从业者（导演、摄影、制片人）进行分镜头方案设计和知识查询。

## 功能概览

本skill提供三大核心功能：

1. **知识查询** - 基于分镜6大核心要点的专业问答
2. **方案设计** - 根据场景描述生成完整分镜头方案
3. **模板工具** - 提供可直接使用的分镜模板和绘制方法

## 6大核心要点

| 序号 | 主题 | 说明 |
|:---:|------|------|
| 1 | 空间设计 | 分镜头要对空间进行详细描摹 |
| 2 | 位置关系 | 人物之间、人物与背景的位置关系必须明确 |
| 3 | 运动关系 | 体现人物在运动中的位置变化 |
| 4 | 画幅与景别 | 明确画幅大小和景别之间的关系 |
| 5 | 明暗关系 | 体现光线如何照亮场景和人物 |
| 6 | 多图配合 | 镜头视角设计图配合二维平面图 |

## 使用方式

### 知识查询

直接问关于分镜的专业问题：
```
"什么是明暗关系？"
"分镜头中如何体现运动关系？"
"宽银幕和普通画幅有什么区别？"
```

### 方案设计

提供场景信息，生成完整分镜方案：
```
我需要为一个汽车品牌广告设计分镜
场景：城市傍晚，车从隧道驶出
时长：30秒
风格：高级、简约、科技感
```

### 模板工具

请求具体的模板或工具：
```
"给我一个分镜表格模板"
"如何画机位图？"
"餐厅谈判场景的空间设计怎么做？"
```

## 核心术语

### 景别

| 景别 | 画面内容 |
|------|----------|
| 远景 (LS) | 整个人物+环境 |
| 全景 (WS) | 整个人物 |
| 中景 (MS) | 膝盖以上 |
| 近景 (MCS) | 胸部以上 |
| 特写 (CU) | 脸部/局部 |

### 光学术语

| 术语 | 含义 |
|------|------|
| 主光 (Key Light) | 最亮的光，决定整体亮度 |
| 补光 (Fill Light) | 填补阴影 |
| 轮廓光 (Rim Light) | 勾勒人物边缘 |
| 光比 | 主光和补光的比例 |

### 画幅

| 比例 | 用途 |
|------|------|
| 1.85:1 | 普通院线 |
| 2.39:1 | 宽银幕 |
| 1.43:1 | IMAX |
| 16:9 | 电视剧/网络 |
