# 影视分镜 Skill

帮助从业者（导演、摄影、制片人）进行分镜头方案设计和知识查询。

## 功能概览

本skill提供三大核心功能：

1. **知识查询** - 基于分镜6大核心要点（空间设计、位置关系、运动关系、画幅景别、明暗关系、多图配合）的专业问答
2. **方案设计** - 根据用户提供的场景描述，生成完整的分镜头方案
3. **模板工具** - 提供可直接使用的分镜模板和绘制方法

## 使用方式

### 方式一：知识查询

直接问关于分镜的专业问题：

- "什么是明暗关系？"
- "分镜头中如何体现运动关系？"
- "宽银幕和普通画幅有什么区别？"

### 方式二：方案设计

提供场景信息，让skill生成完整分镜方案：

```
我需要为一个汽车品牌广告设计分镜
场景：城市傍晚，车从隧道驶出，阳光从后方照射
时长：30秒
产品：新能源汽车
风格：高级、简约、科技感
```

### 方式三：模板工具

请求具体的模板或工具：

- "给我一个分镜表格模板"
- "如何画机位图？"
- "餐厅谈判场景的空间设计怎么做？"

## 核心知识体系

### 1. 空间设计
电影是空间艺术，分镜头不仅针对人物，也要对空间进行详细的描摹和刻画。

**关键要点：**
- 场景的整体形状（室内/室外、形状）
- 空间的大小、高度
- 门窗位置（决定光线来源和人物进出）
- 主要物体/家具（决定功能和年代）
- 环境细节（墙上画作、桌上物品等）

### 2. 位置关系
人物之间、人物和背景之间的位置关系必须明确。

**关键要点：**
- 谁在左/谁在右
- 谁离镜头近/谁离镜头远
- 人物和背景的相对位置

### 3. 运动关系
要体现人物在运动中的位置变化，这是区分漫画和电影的关键。

**关键要点：**
- 人物的移动路径
- 演员走位时机
- 摄影机运动方式

### 4. 画幅与景别
明确画幅大小和景别之间的关系。

**常见画幅：**
| 比例 | 用途 |
|------|------|
| 1.85:1 | 普通院线 |
| 2.39:1 | 宽银幕史诗 |
| 1.43:1 | IMAX |
| 16:9 | 电视剧/网络 |

**景别：**
| 景别 | 内容 |
|------|------|
| 远景 | 整个人物+环境 |
| 全景 | 整个人物 |
| 中景 | 膝盖以上 |
| 近景 | 胸部以上 |
| 特写 | 脸部/局部 |

### 5. 明暗关系
体现光线如何照亮场景和人物。

**专业术语：**
| 术语 | 含义 |
|------|------|
| 主光 (Key Light) | 最亮的光，决定整体亮度 |
| 补光 (Fill Light) | 填补阴影 |
| 轮廓光 (Rim Light) | 勾勒人物边缘 |
| 光比 | 主光和补光的比例 |

### 6. 多图配合
镜头视角设计图要配合二维平面图。

**三图合一：**
- 灯位图：灯放在哪
- 机位图：摄影机在哪
- 调度图：演员怎么走位

## 交互流程

### 启动欢迎语

```
🎬 影视分镜 Skill

我是你的分镜助手，帮助你：
• 查询分镜专业知识（空间设计、机位、景别、光线等）
• 设计完整的分镜头方案
• 提供分镜模板和工具

请告诉我你需要什么：
1. 问问题（如"什么是明暗关系？"）
2. 设计分镜（如"帮我设计一个汽车广告分镜"）
3. 获取模板（如"给我一个分镜表格模板"）
```

### 场景设计方案

当用户提供场景时，按以下流程处理：

**Step 1: 确认信息**
- 项目类型：_____
- 时长：_____
- 风格：_____
- 场景：_____

**Step 2: 生成方案**
基于信息生成完整分镜表格，包含：
- 镜头序列（镜号+景别+运动+时长）
- 空间描述
- 机位建议
- 光线设计

**Step 3: 可视化建议**
- 绘制机位图
- 标注走位图
- 绘制光位图

### 知识查询响应

按6大核心要点组织回答：
1. 空间设计
2. 位置关系
3. 运动关系
4. 画幅与景别
5. 明暗关系
6. 多图配合

### 模板工具响应

提供可直接复制使用的模板：
- 分镜表格
- 机位图符号
- 光位图符号
- 检查清单
