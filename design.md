# 方案设计模块

## 概述

本模块根据用户提供的场景描述，生成完整的分镜头方案。

---

## 输入信息模板

用户提供场景时，建议包含以下信息：

```
## 场景基本信息
- 项目类型：_____
- 时长：_____
- 风格：_____

## 核心场景描述
- 场景：_____
- 人物：_____
- 动作：_____

## 特殊要求
- 品牌元素：_____
- 色调：_____
- 音乐/音效：_____
```

## 方案生成流程

### Step 1: 分析场景元素

1. **场景类型**：室内/室外/混合
2. **时间**：日/夜/晨/昏
3. **空间特点**：开阔/封闭/复杂
4. **人物数量**：单人/多人/群像
5. **核心动作**：静态对话/运动/产品展示

### Step 2: 确定镜头节奏

根据时长分配镜头数量：

| 时长 | 建议镜头数 | 节奏 |
|------|------------|------|
| 15s  | 6-10个    | 快节奏，每个镜头1-2秒 |
| 30s  | 12-20个   | 中等节奏 |
| 60s  | 20-40个   | 慢节奏，可有长镜头 |

### Step 3: 设计镜头序列

1. **建立镜头 (Establishing)** - 交代环境
2. **关系镜头 (Relation)** - 人物关系
3. **动作镜头 (Action)** - 核心动作
4. **特写镜头 (Close-up)** - 产品/表情
5. **结尾镜头 (Closing)** - 收尾/Logo

### Step 4: 输出完整方案

## 机位设计指南

### 常用机位术语

| 机位 | 说明 |
|------|------|
| 正拍 | 摄影机正对被摄体 |
| 侧拍 | 摄影机与被摄体成90度 |
| 侧后拍 | 摄影机在被摄体侧后方45度 |
| 逆光 | 摄影机逆光方向 |
| 顺光 | 摄影机顺光方向 |

### 商业广告常用机位

**产品展示：**
- 0度低角度：产品高大感
- 45度侧光：立体感
- 顶拍：独特视角

**人物：**
- 略低于眼睛：权威感
- 略高于眼睛：亲和感
- 侧光：戏剧性

---

## 光线设计指南

### 商业广告常见布光风格

**高端简约：**
- 主光：柔和顶光
- 补光：低强度正面光
- 无轮廓光
- 整体：低对比、清爽

**戏剧张力：**
- 主光：强侧光
- 补光：极弱或无
- 轮廓光：强
- 整体：高对比、戏剧

**自然日常：**
- 主光：窗户光
- 补光：环境光
- 无人工轮廓光
- 整体：自然、低调

### 不同产品的光线建议

| 产品类型 | 光线风格 |
|----------|----------|
| 汽车 | 轮廓光强烈、金属质感 |
| 化妆品 | 柔和正面光、皮肤质感 |
| 电子产品 | 蓝色调、冷光、科技感 |
| 食品 | 暖色调、侧光、立体感 |

## 成本控制建议

### 减少拍摄成本的技巧

1. **减少转场** - 同一场景多角度拍摄
2. **固定机位** - 减少设备调动时间
3. **自然光为主** - 减少灯光设备
4. **日夜景通吃** - 早晨/黄昏可模拟多时段

### 需要增加成本的情况

1. **复杂运动** - 需要额外设备和彩排
2. **特殊视角** - 需要吊臂、摇臂、无人机
3. **大量群演** - 协调成本高
4. **特效合成** - 绿幕拍摄+后期
