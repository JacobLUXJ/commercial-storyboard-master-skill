# 核心知识模块

## 概述

本模块提供分镜专业的知识查询功能，基于6大核心要点：空间设计、位置关系、运动关系、画幅景别、明暗关系、多图配合。

---

## 1. 空间设计

### 1.1 空间为什么重要？

电影是空间艺术，与文学、戏剧最大的区别在于：
- **文学**：通过文字描述，读者自己想象空间
- **戏剧**：舞台固定，观众一览无余
- **电影**：镜头可以任意选择角度、距离、范围来呈现空间

电影的情绪、氛围、年代感，全部要通过空间来传达。

### 1.2 分镜头中空间要画什么？

| 要画的内容 | 目的 |
|------------|------|
| 场景整体形状 | 室内？室外？长方形？正方形？ |
| 空间大小、高度 | 宽阔还是狭窄？层高多少？ |
| 门窗位置 | 光从哪来？人物怎么进出？ |
| 主要物体/家具 | 决定场景功能和年代感 |
| 环境细节 | 墙上有没有画？桌子上有什么？ |

### 1.3 空间如何影响镜头调度

- 想表现"压抑" → 用低角度拍高天花板
- 想表现"孤独" → 让角色站在空旷空间边缘
- 想表现"时间流逝" → 让光线在空间中移动

### 1.4 空间如何影响成本

| 空间复杂度 | 意味着什么 |
|------------|------------|
| 实景拍摄 | 可能无法收音、有人围观、天气影响 |
| 搭景 | 要花钱搭，但完全可控 |
| 复杂历史场景 | 可能需要大量置景道具 |

## 2. 位置关系

### 2.1 什么是位置关系？

电影画面不是平的，它有纵深感：
- A和B谁在左？谁在右？
- 谁离镜头近？谁离镜头远？
- 人物和背景（墙、窗、家具）之间的相对位置

### 2.2 为什么要明确位置关系？

**直接影响调度精确性**
- 摄影师不知道机位该摆在哪
- 灯光师不知道光从哪个方向打
- 演员不知道站哪
- 结果：拍摄现场一团乱，现场临时决定，时间成本飙升

**决定镜头能不能实现**
比如：导演想拍一个"从A肩后看B"的眼神镜头，但如果A和B的位置站反了，这个镜头就拍不了。

### 2.3 位置关系如何影响情绪

| 位置关系 | 传达的情绪 |
|----------|------------|
| 面对面，紧挨着 | 亲密、信任 |
| 面对面，中间有桌子 | 对抗、谈判 |
| 背对背，各坐一边 | 疏远、冷战 |
| 一个坐着，一个站着 | 权力不对等 |

### 2.4 如何画位置关系

**方法一：平面图标注**
画一个简单的场景俯视图，标注：
- 人物位置（用圆圈代表）
- 摄影机位置（用方框代表）
- 门窗位置

**方法二：机位图**
直接画摄像机在哪里，拍摄角度是什么

**方法三：文字标注**
在分镜旁边写：
- "A在画面左侧，B在画面右侧"
- "A前景，B后景"
- "A离镜头2米，B离镜头5米"

## 3. 运动关系

### 3.1 为什么运动关系重要？

漫画是静态的，电影是动态的。漫画只需要画一个"决定性瞬间"，但电影要记录整个运动过程。

这是区分漫画和电影的关键。

### 3.2 分镜头中要体现什么

| 静态信息 | 动态信息 |
|----------|----------|
| A站在房间左边 | A从门口走向窗户（路径） |
| B坐在椅子上 | B突然站起来（时机） |
| 摄影机在墙角 | 摄影机缓慢推向B（运动方式） |

### 3.3 如何体现运动

**方法一：箭头标注**
```
A ———→ 窗户
   慢走
```

**方法二：编号+路径图**
```
镜头1：A从门口走向窗户（5秒）
镜头2：A停下转身（2秒）
镜头3：推向A表情（3秒）
```

**方法三：时间轴**
```
0s  2s  5s  7s
|   |   |   |
●———●———●
走  停  转身
```

### 3.4 运动关系如何影响成本

| 情况 | 意味着什么 |
|------|------------|
| 复杂运动（追逐、舞蹈） | 需要大量彩排 |
| 多人物交叉运动 | 需要副导演精确协调 |
| 运动路线不明确 | 现场试多种方案 = 超时 |

## 4. 画幅与景别

### 4.1 常见画幅

| 比例 | 用途 | 特点 |
|------|------|------|
| 1.85:1 | 普通院线（最常见） | 均衡 |
| 2.39:1 | 宽银幕史诗大片 | 横向信息多，适合大场面 |
| 1.43:1 | IMAX | 信息量巨大，适合细节展示 |
| 16:9 | 电视剧/网络 | 标准屏幕 |
| 4:3 | 老式电视 | 基本淘汰 |

### 4.2 景别详解

| 景别 | 画面内容 | 适合场景 |
|------|----------|----------|
| 远景 (LS) | 整个人物+环境 | 交代场景、营造氛围 |
| 全景 (WS) | 整个人物 | 展示动作、空间感 |
| 中景 (MS) | 膝盖以上 | 对话、日常 |
| 近景 (MCS) | 胸部以上 | 情绪、表情 |
| 特写 (CU) | 脸部/局部 | 强调、细节 |

### 4.3 画幅如何影响镜头设计

**宽银幕 (2.39:1)**
- 横向信息多
- 适合横向运动、大场面
- 人物在画面中显得小

**普通画幅 (1.85:1)**
- 画面更紧凑
- 人物更突出

**IMAX (1.43:1)**
- 信息量巨大
- 适合细节展示

### 4.4 焦段与视角

| 焦段 | 视角特点 |
|------|----------|
| 24mm | 广角，视野宽，透视强 |
| 35mm | 略广角，兼顾环境 |
| 50mm | 接近人眼视角，自然 |
| 85mm | 中长焦，压缩空间 |
| 135mm | 长焦，浅景深，压缩感强 |

## 5. 明暗关系

### 5.1 什么是明暗关系？

明暗关系 = 光怎么照在场景和人物上
- 哪里亮？
- 哪里暗？
- 光从哪个方向来？

### 5.2 为什么要画明暗关系？

**决定布光方案**
摄影指导和灯光师需要知道：
- 主光从哪来？
- 补光要打多亮？
- 需要轮廓光吗？

**影响人物情绪**

| 光线 | 情绪 |
|------|------|
| 柔和正面光 | 温暖、友善 |
| 硬侧光 | 紧张、戏剧性 |
| 底光（从下往上） | 恐怖、神秘 |
| 柔光箱直接照射 | 纪录片、真实 |

**影响画面质感**
- 好莱坞黄金时代：柔和分层光
- 黑色电影：强烈明暗对比
- 文艺片：自然光、低对比

### 5.3 核心光学术语

| 术语 | 英文 | 含义 |
|------|------|------|
| 主光 | Key Light | 最亮的光，决定整体亮度 |
| 补光 | Fill Light | 填补阴影，减低对比 |
| 轮廓光 | Rim Light | 勾勒人物边缘，分离主体和背景 |
| 底光 | Bottom Light | 从下往上照，制造恐怖感 |
| 光比 | Lighting Ratio | 主光和补光的比例 |
| 软光 | Soft Light | 柔光箱/窗户光 |
| 硬光 | Hard Light | 聚光灯/阳光 |

### 5.4 如何画明暗

**方法一：光位图符号**
```
    ○ ← 主光（左上方）
    /
    ● ← 人物
    /
    ← 相机
```

**方法二：标注+文字**
```
[镜头3] 夜景室内
- 主光：窗户光（右侧）
- 补光：弱柔光箱（正面）
- 轮廓光：无
- 整体氛围：暗淡、神秘
```

## 6. 多图配合

### 6.1 什么是镜头视角设计图？

- 机位图（Camera Position）
- 运镜图（Camera Movement）

### 6.2 什么是二维平面设计图？

- 场景平面图
- 置景施工图
- 灯位图
- 走位图

### 6.3 为什么要配合使用？

**三维无法完全表达**
- 仰拍/俯拍 → 画不出来
- 旋转运镜 → 画不出来
- 复杂的空间关系 → 很难画

解决方法：配合二维平面图一起看。

### 6.4 三图合一

书里建议把三个东西合并到一张图：

| 图 | 内容 |
|----|------|
| 灯位图 | 灯放在哪 |
| 机位图 | 摄影机在哪 |
| 调度图 | 演员怎么走位 |

合并成一张 **"摄影灯光位调度综合图"**

### 6.5 实际操作步骤

**步骤1：画场景平面图**
```
[门] [窗] [窗户]
[门]  [桌]  [摄]
```

**步骤2：标注机位+灯位**
```
[门] [灯1] [窗] [灯2]
[门]   (A)  /  (B)
      摄影机1 摄影机2
```

**步骤3：标注调度**
```
[门] [灯1] [窗] [灯2]
[门]   (A) 走  (B)
      摄影机1 摄影机2
```

## 常见问题FAQ

**Q: 分镜头需要多详细？**
A: 取决于项目规模。广告通常每5秒一个镜头，电影可能更密。关键是传达清晰 Enough for the crew to work from。

**Q: 手绘还是软件？**
A: 都可以。专业软件有Storyboard Pro、Shot Designer，手绘可用Procreate。关键是清晰表达意图。

**Q: 分镜和剧本有什么区别？**
A: 剧本是文字描述，分镜是视觉化呈现。分镜更关注：怎么拍、在哪拍、拍多久。

**Q: 商业广告分镜要注意什么？**
A: 1) 产品必须有清晰展示 2) 时长严格控制 3) 节奏要快 4) 品牌元素突出
