# 影视分镜 Skill

帮助从业者进行分镜头方案设计和知识查询。

## 功能概览

1. **知识查询** - 基于分镜6大核心要点的专业问答
2. **方案设计** - 根据场景描述生成完整分镜头方案
3. **模板工具** - 提供可直接使用的分镜模板

## 6大核心要点

### 1. 空间设计

电影是空间艺术，分镜头不仅针对人物，也要对空间进行详细描摹。

**要画什么：**
| 内容 | 目的 |
|------|------|
| 场景整体形状 | 室内/室外、形状 |
| 空间大小、高度 | 宽阔/狭窄 |
| 门窗位置 | 光源方向、人物进出 |
| 主要物体/家具 | 功能和年代感 |
| 环境细节 | 墙上画作、桌上物品等 |

**空间如何影响镜头：**
- 表现"压抑" → 低角度拍高天花板
- 表现"孤独" → 让角色站在空旷空间边缘
- 表现"时间流逝" → 让光线在空间中移动

### 2. 位置关系

人物之间、人物和背景之间的位置关系必须明确。

**什么是位置关系：**
- 谁在左/谁在右？
- 谁离镜头近/谁离镜头远？
- 人物和背景的相对位置

**不明确的代价：**
- 摄影师不知道机位该摆在哪
- 灯光师不知道光从哪个方向打
- 演员不知道站哪
- 结果：现场一团乱，时间成本飙升

**位置与情绪：**
| 位置关系 | 情绪 |
|----------|------|
| 面对面，紧挨着 | 亲密、信任 |
| 面对面，中间有桌子 | 对抗、谈判 |
| 背对背，各坐一边 | 疏远、冷战 |
| 一个坐着，一个站着 | 权力不对等 |

### 3. 运动关系

要体现人物在运动中的位置变化。这是区分漫画和电影的关键。

**漫画 vs 电影：**
- 漫画：静态的，只需画"决定性瞬间"
- 电影：动态的，要记录整个运动过程

**如何标注运动：**
```
箭头标注：
A ———→ 窗户
   慢走

编号+路径：
镜头1：A从门口走向窗户（5秒）
镜头2：A停下转身（2秒）

时间轴：
0s  2s  5s  7s
|   |   |   |
●———●———●
走  停  转身
```

### 4. 画幅与景别

明确画幅大小和景别之间的关系。

**常见画幅：**
| 比例 | 用途 |
|------|------|
| 1.85:1 | 普通院线（最常见） |
| 2.39:1 | 宽银幕史诗大片 |
| 1.43:1 | IMAX |
| 16:9 | 电视剧/网络 |

**景别：**
| 景别 | 画面内容 | 适合场景 |
|------|----------|----------|
| 远景 (LS) | 整个人物+环境 | 交代场景 |
| 全景 (WS) | 整个人物 | 展示动作 |
| 中景 (MS) | 膝盖以上 | 对话/日常 |
| 近景 (MCS) | 胸部以上 | 情绪 |
| 特写 (CU) | 脸部/局部 | 强调 |

**焦段与视角：**
| 焦段 | 特点 |
|------|------|
| 24mm | 广角，视野宽 |
| 50mm | 接近人眼 |
| 85mm | 压缩，空间浅 |

### 5. 明暗关系

体现光线如何照亮场景和人物。

**专业术语：**
| 术语 | 含义 |
|------|------|
| 主光 (Key Light) | 最亮的光，决定整体亮度 |
| 补光 (Fill Light) | 填补阴影，减低对比 |
| 轮廓光 (Rim Light) | 勾勒人物边缘 |
| 光比 | 主光和补光的比例 |
| 软光 | 柔光箱/窗户光 |
| 硬光 | 聚光灯/阳光 |

**光线与情绪：**
| 光线 | 情绪 |
|------|------|
| 柔和正面光 | 温暖、友善 |
| 硬侧光 | 紧张、戏剧性 |
| 底光（从下往上） | 恐怖、神秘 |

### 6. 多图配合

镜头视角设计图要配合二维平面图。

**三图合一：**
- 灯位图：灯放在哪
- 机位图：摄影机在哪
- 调度图：演员怎么走位

合并成 **"摄影灯光位调度综合图"**

## 方案设计流程

### Step 1: 分析场景元素
- 场景类型：室内/室外
- 时间：日/夜/晨/昏
- 人物数量：单人/多人

### Step 2: 确定镜头节奏
| 时长 | 建议镜头数 |
|------|------------|
| 15s | 6-10个 |
| 30s | 12-20个 |
| 60s | 20-40个 |

### Step 3: 设计镜头序列
1. 建立镜头 - 交代环境
2. 关系镜头 - 人物关系
3. 动作镜头 - 核心动作
4. 特写镜头 - 产品/表情
5. 结尾镜头 - 收尾/Logo

### Step 4: 输出分镜表格

| 镜号 | 景别 | 运动 | 机位 | 时长 | 空间描述 | 调度 | 光线 | 备注 |
|------|------|------|------|------|----------|------|------|------|
| 1 | 远景 | 固定 | 正侧远景 | 3s | 城市街道全景 | 车从画面右侧驶入 | 傍晚自然光 | 建立环境 |
| 2 | 中景 | 推镜 | 侧后方 | 4s | 车侧面跟随 | 车保持匀速 | 逆光轮廓光 | 展示产品 |

### 机位图符号

```
    ○ = 人物
    □ = 摄影机
    △ = 重要物体
    → = 人物运动路径
```

### 光位图符号

```
    ☀ = 主光 (Key Light)
    ☁ = 补光 (Fill Light)
    ◐ = 轮廓光 (Rim Light)
    ▼ = 底光
```

3. □ 运动路径是否标注了？

6. □ 机位是否清晰？
7. □ 时长是否合理？

## 成本控制要点

### 减少成本的技巧
- 减少转场：同一场景多角度拍摄
- 固定机位：减少设备调动
- 自然光为主：减少灯光设备

### 需要增加成本的情况
- 复杂运动：需要额外设备和彩排
- 特殊视角：需要吊臂、无人机
- 大量群演：协调成本高
