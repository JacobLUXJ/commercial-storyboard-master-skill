# 模板工具模块

## 概述

本模块提供可直接使用的分镜模板、绘制方法说明和实用工具。

---

## 分镜表格模板

### 标准模板（Excel格式）

```
┌──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┐
│ 镜号 │ 景别 │ 运动 │ 机位 │ 时长 │ 空间描述       │ 调度      │ 备注 │
├──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│  1   │ 远景 │ 固定 │ 正拍 │  3s  │ 城市全景       │ 车的运动   │     │
│  2   │ 中景 │ 推镜 │ 侧后 │  4s  │ 车侧跟随       │ 同上      │     │
└──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┘
```

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 镜号 | 镜头序号 | 1, 2, 3... |
| 景别 | 镜头大小 | 远景/全景/中景/近景/特写 |
| 运动 | 摄影机运动 | 固定/推/拉/摇/移/跟/升降 |
| 机位 | 拍摄角度 | 正拍/侧拍/逆光/俯拍/仰拍 |
| 时长 | 镜头时长 | 2s, 3s, 5s |
| 空间描述 | 场景细节 | 室内咖啡馆、落地窗、逆光 |
| 调度 | 演员走位 | A从左入画、B站起 |
| 备注 | 特殊说明 | 产品露出、Logo等 |

## 机位图绘制指南

### 基本符号

```
┌─────────────────────────────────┐
│                                 │
│    ○ = 人物                     │
│    □ = 摄影机                   │
│    △ = 重要物体                 │
│    ╱ = 摄影机朝向               │
│    → = 人物运动路径             │
│    ⊕ = 摄影机位置(俯视)        │
│                                 │
└─────────────────────────────────┘
```

### 机位图示例：餐厅场景

```
              [窗] [窗] [窗]
               │     │
        ┌──────┴──────┴──────┐
        │                    │
        │    ○A      ○B      │  ← 人物位置
        │   (餐厅桌)         │
        │                    │
   ╱═══╱     □              │  ← 机位1：侧拍
        │         ╱═════╲   │
        │                □   │  ← 机位2：过肩
        └────────────────────┘

        [灯1]      ○      [灯2]
```

### 常用机位图符号

```
俯视视角（最常用）：

    ┌──────────────────┐
    │     [窗]         │
    │        │         │
    │   ○A───┼───○B    │
    │        │         │
    │   □1   │    □2   │
    │    ╲   │   ╱     │
    │     ╲  │  ╱      │
    │      ╲ │ ╱       │
    │       ○│○         │
    └──────────────────┘

    ○ = 人物  □ = 摄影机  ╱ = 朝向
```

## 光位图绘制指南

### 光线符号

```
    ☀ = 主光 (Key Light)
    ☁ = 补光 (Fill Light)
    ◐ = 轮廓光 (Rim Light)
    ▼ = 底光 (Bottom Light)
    ◎ = 反射光
    ✷ = 散射光
```

### 光位图示例：室内人像

```
           ☀ 主光(窗光)
             ╱
           ╱
         ╱
    [窗]╱          ◐ 轮廓光
       ╱
      ╱
    ──╱──── ────── ──────
     ╱     ○        ◐
    ╱
   ╱            ☁ 补光(弱)
  ☀
         □ 摄影机
```

### 布光方案速查

**方案1：经典三点布光**
```
    ☀ (主光45度)
       ╲
        ╲
         ○ (人物)
        ╱
    ☁ (补光弱) ◐ (轮廓光)
```

**方案2：窗光效果**
```
    [窗]
      │
      │ ☀
      ▼
       ○
```

**方案3：羽化蝴蝶光**
```
    ☀ (高位)
       │
    ───┼───
       ○
```

## 走位图绘制指南

### 走位符号

```
    ○ = 演员当前位置
    → = 运动路径
    1→2→3 = 运动顺序
    ⊕ = 起点
    ◎ = 终点
    ⊙ = 中间位置
```

### 走位图示例：追逐场景

```
    [门]                      [窗]
      │                         │
      │   ┌─────┐              │
      │   │ 桌  │              │
      │   └─────┘              │
      │                         │
      │    ⊕→⊙→◎              │
      │    A追逐B              │
      │                         │
    □1 ╲                      ╱ □2
        ╲                    ╱
         ╲    ○B  ○A        ╱
          ╲  │    │        ╱
           ╲ │    │       ╱
            ○│────┘      □3
              摄影机位置
```

### 调度说明模板

```
[调度说明]
- A：从门口进入，沿左侧走到窗边
- B：坐在桌旁，A靠近时站起
- 时机：A到达窗边时，B站起
- 摄影机1：固定拍全景
- 摄影机2：A近景时推镜
```

## 检查清单

### 分镜自检10问

1. □ 空间是否画清楚了？（场景、门窗、主要物体）
2. □ 人物位置关系是否明确？（谁左谁右、谁前谁后）
3. □ 运动路径是否标注了？（箭头、编号）
4. □ 画幅和景别是否匹配？
5. □ 光线方向是否标注了？
6. □ 机位是否清晰？
7. □ 时长是否合理？
8. □ 镜头衔接是否流畅？
9. □ 产品展示是否足够？
10. □ 预算是否可控？
